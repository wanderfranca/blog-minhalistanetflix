Import Dummy data for themes developed by Template Sell. 
== Description ==
A easy plugin to import dummy data for themes by Template Sell.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for Template Sell Demo Importer
3. Activate Template Sell Demo Importer plugin from your Plugins page. 
4. Go to the Appearance -> Import Demo Data 
Note: Make sure, One click demo import plugin is installed and activated first.

= From WordPress.org =

1. Download templatesell-demo-importer.
2. Upload the 'templatesell-demo-importer' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Template Sell Demo Importer from your Plugins page. 
Note: Make sure, One click demo import plugin is installed and activated first.

== Frequently Asked Questions ==
1. Where I can see the Settings? 
Check all the available options inside Appearance > Import Demo Data
