== Description ==

Amigo is a modern, responsive WordPress theme that allows you to create extensive reviews for multiple content including games, travel destinations, films, technology etc.

= Theme Features Usage =

After installing theme download and activate these plugins: 

Kirki Toolkit - https://wordpress.org/plugins/kirki/ - theme options toolkit
WP Review - https://wordpress.org/plugins/wp-review/ - WP Review plugin is an easy yet powerful way to review content easily (if you dont need it, dont install it)

= Theme Features Usage =
All available options can be used from Appearance->Customize

Get theme instruction at http://demo.themes4wp.com/documentation/category/amigo/
